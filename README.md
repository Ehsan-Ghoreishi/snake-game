# Snake Game

A classic Snake game implementation built with Python and Pygame.

## Description

This is a traditional Snake game where the player controls a snake that grows longer as it eats food. The objective is to eat as much food as possible without colliding with the walls or the snake's own body.

## Features

- Classic snake gameplay mechanics
- Smooth snake movement with arrow key controls
- Food spawning at random locations
- Snake growth system
- Game over detection (wall collision and self-collision)
- Restart or quit options after game over

## Requirements

- Python 3.9 or higher
- Pygame library

## Installation

1. Make sure you have Python installed on your system
2. Install Pygame using pip:

```bash
pip install pygame
```

## How to Run

Run the game using Python:

```bash
python snake_game.py
```

## Controls

- **Arrow Keys**: Control snake direction
  - ↑ Up Arrow: Move up
  - ↓ Down Arrow: Move down
  - ← Left Arrow: Move left
  - → Right Arrow: Move right

## Game Over Options

When the game ends:
- Press **C**: Play again
- Press **Q**: Quit the game

## Game Rules

1. Use arrow keys to control the snake's direction
2. Eat the green food blocks to grow longer
3. Avoid hitting the walls (screen boundaries)
4. Avoid colliding with your own snake body
5. The game ends when you hit a wall or yourself

## Game Settings

- **Display Size**: 800x600 pixels
- **Snake Speed**: 20 FPS
- **Snake Block Size**: 10x10 pixels

## Colors

- Background: Blue
- Snake: Black
- Food: Green
- Game Over Text: Red
